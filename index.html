Featured Projects

Things I've built — some out of curiosity, some out of necessity

Lil Baymax

The Problem
Healthcare monitoring in low-resource settings often means someone physically checking on a patient. There's no continuous, mobile way to track vitals and alert caregivers remotely.

What I Built
A self-balancing two-wheeled robot that reads live vitals — temperature, SpO₂, heart rate, blood pressure — and sends GSM alerts when values go out of range. PID-tuned balance on ESP32, 3D-printed chassis.

ESP32 PID Control Healthcare IoT 3D Printing
GitHub

Chokh-e-Dekha

The Problem
People had no practical way to officially report broken roads or civic issues. And even where channels existed, there was no visibility into what happened after — no accountability, no transparency.

What I Built
A civic reporting platform where residents log issues with location tags, and admins triage, assign, and close them through a dashboard. Every report has a status trail — so citizens can actually see if anything happened.

Laravel MySQL Google Maps API Civic Tech
GitHub

TravelEase

The Problem
University group project — the brief was to build a real booking platform, not a mockup. Most teams build something that looks like a product but doesn't function like one.

What I Built
Led the team to build a fully working booking system with Stripe payments, real-time availability, admin analytics, and a 90+ Lighthouse score. Designed the database schema and led architecture decisions throughout.

Laravel Livewire PostgreSQL Stripe
GitHub

4-DOF Robotic Arm

The Problem
University robotics project — building a precise, controllable arm with no expensive components and no off-the-shelf controller.

What I Built
A four-axis arm controlled wirelessly via MPU6050 gyroscope and nRF24L01 radio. Inverse kinematics on STM32, 3D-printed links, ±1mm repeatability. PCB designed from scratch in KiCad.

C++ STM32 KiCad 3D Printing
GitHub

Pharmacy Management System

What I Built
A comprehensive management system covering medicine stock, expiry tracking, financial records, and reporting. Built as a prototype for Xactidea — they refined it and delivered it to a client.
